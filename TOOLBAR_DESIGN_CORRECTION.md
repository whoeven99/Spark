## 🎨 ToolBar 组件详细设计

### ToolBar 外观
```
┌─────────────────────────────────────────────────────────────┐
│ [选择对象] | [参考文档] [数据源] [多媒体] | [约束] [清空] │
└─────────────────────────────────────────────────────────────┘
```

### 各按钮功能

#### 1. **选择对象** 按钮
```
显示: [选择对象] 或 [✓ 42个] (已选)
点击: 打开模态框选择对象
样式: 已选时为 primary 类型，未选时为 default
```

#### 2. **参考文档** 按钮  
```
显示: [参考文档] 或 [📎 3] (已上传3个)
点击: 打开 Popover 显示文档上传/管理
样式: 已上传时为 primary 类型
支持: 拖拽上传、删除
```

#### 3. **数据源** 按钮
```
显示: [数据源] 或 [✓ 数据源] (已上传)
点击: 打开数据源上传对话框
样式: 已上传时为 primary 类型
限制: 最多上传 1 个文件
```

#### 4. **多媒体** 按钮
```
显示: [多媒体] 或 [🖼️ 2] (已上传2个)
点击: 打开 Popover 显示多媒体上传/管理
样式: 已上传时为 primary 类型
支持: 图片、视频、音频，最多 5 个
```

#### 5. **约束** 按钮
```
显示: [约束⚙️]
点击: 打开模态框设置约束条件
功能:
  • 最大长度设置
  • 最小长度设置
  • 保留内容选项 (HTML、链接、元数据)
```

#### 6. **清空** 按钮 (右对齐)
```
显示: [清空] (危险按钮，红色)
点击: 清空所有上下文信息
确认: 可选 confirm 对话框
```

## 🔧 实现细节

### ToolBar 组件结构
```
├── 水平布局 (Flexbox)
├── 按钮组
│   ├── 选择对象 + Modal
│   ├── 参考文档 + Popover
│   ├── 数据源 + Modal/Popover
│   ├── 多媒体 + Popover
│   ├── Divider 分隔线
│   ├── 约束 + Modal
│   └── 清空按钮 (右对齐)
└── 响应式 (flexWrap: wrap)
```

### 工具栏状态反馈
```
未选择状态: [选择对象] [参考文档] ...
已选择状态: [✓ 42个] [📎 3] ...
          (按钮变为 primary 类型，显示数量)
```

### Popover vs Modal 的选择
```
Popover (弹窗):
  - 参考文档: 快速查看和管理
  - 多媒体: 快速上传和删除

Modal (模态):
  - 选择对象: 需要复杂的表格选择交互
  - 约束条件: 需要多个表单控件
  - 数据源: 可用 Popover (后续优化)
```

## 🔄 与之前设计的区别

| 方面 | 之前 | 现在 |
|------|------|------|
| ToolBar 位置 | 左侧导航栏 (280px 列) | 对话框上方 (水平工具栏) |
| 布局方式 | 三列布局 | 两列布局 (中央 + 右侧) |
| ToolBar 样式 | Card 卡片 | 按钮组 + Popover/Modal |
| 视觉占用 | 占用 25% 左侧空间 | 占用顶部最小空间 |
| 对话区域 | 中央偏小 | 充分利用空间 |
| 右侧任务列表 | 固定 320px | 固定 340px |

---

## ✨ 改进点

1. **空间利用更高效** - 不浪费左侧导航空间
2. **视觉层级清晰** - ToolBar 在对话框上方，逻辑清晰
3. **交互更紧凑** - 所有工具集中在顶部，一目了然
4. **对话区域更宽敞** - 充分显示对话历史和卡片
5. **响应式更好** - 工具栏可以自动换行适应不同宽度
